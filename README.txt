Steam Deck ElvUI Installer
==========================

This addon installs a Steam Deck-optimized layout for ElvUI and includes:

- Custom action bar and chat settings
- ConsolePort support (controller-friendly)
- ConsoleBags support (disables ElvUI bags)
- Plater profile import
- Auto-detects bag addon conflicts

Install Instructions:
1. Copy this folder into your `_retail_/Interface/AddOns/` directory.
2. Launch WoW and run `/sdui install` to apply the layout.
